# Unity 3D Horror Zombie Game

**Unity Horror Zombie Game** is a 3D first-person horror game built in **Unity**. The player navigates through a spooky environment, fights approaching zombies, and collects treasure chests to win.

---

## Gameplay Features

- **First-Person Shooter (FPS) Mechanics**
  - Move using **WASD** keys
  - Look around using **Mouse**
  - Shoot enemies using **Mouse Click**

- **Enemies**
  - Zombies that approach the player
  - AI-based movement towards player

- **Treasure Chests**
  - Collectable objects placed in the map
  - Completing objectives by collecting chests can win the game

- **Weapons**
  - Firearm mechanics with ammo and shooting animations

- **Win/Lose Conditions**
  - Lose if player’s health reaches zero
  - Win if all treasure chests are collected

---

## Tech Stack

- **Engine:** Unity 3D (2019/2020/2021 or later)
- **Language:** C#
- **Input Controls:** Keyboard + Mouse
- **Assets:** Free or custom 3D models, zombie models, environment, and sounds
